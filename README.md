# DoseVision (clean)

Next.js 14 + TypeScript + Tailwind. Light gray UI with navy accents.
Routes:
- /  (Home summary + highlights)
- /optimize  (tabs: Dose Ordering, Schedule, Vendors, Health Insurance, Dose Credits)
- /regulatory (tabs: Daily Area Survey, Weekly Area Survey, Seal Source Inventory, Tracer Check In/Out, Patient Dose Log, Hot Lab Instruments, Dosimeter Tracker, Waste Management, To Do List)
- /schedule (30 sample rows + CSV export)

## Local
npm i
npm run dev

## Build
npm run build

## Deploy (Vercel)
Import the GitHub repo. Framework preset: Next.js.
